대한민국 헌법 제27조 제2항

"군인 또는 군무원이 아닌 국민은 대한민국의 영역안에서는 중대한 군사상 기밀ㆍ초병ㆍ초소ㆍ유독음식물공급ㆍ포로ㆍ군용물에 관한 죄중 법률이 정한 경우와 비상계엄이 선포된 경우를 제외하고는 군사법원의 재판을 받지 아니한다."
🪖
군인 또는 군무원이 아닌 국민은
군사 군(軍) + 사람 인(人) (군인)
군사 군(軍) + 힘쓸 무(務) + 인원 원(員) (군무원)
군인이나 군무원을 제외한, 군대에 속하지 않은 모든 일반 국민을 뜻합니다.
🏛️
...경우를 제외하고는 군사법원의 재판을 받지 아니한다
군사 군(軍) + 법 법(法) + 뜰 원(院) (군사법원)
마를 재(裁) + 판단할 판(判) (재판)
우리 같은 일반 국민은 어떤 잘못을 저질렀더라도, 군인들을 재판하는 특별한 법원인 군사법원이 아니라, 일반 법원에서 판사님에게 재판을 받아야 합니다.
🚫
대한민국의 영역안에서는 중대한 군사상 기밀·초병·초소·유독음식물공급·포로·군용물에 관한 죄중 법률이 정한 경우와
무거울 중(重) + 클 대(大) (중대)
군사 군(軍) + 일 사(事) (군사상)
기밀 기(機) + 비밀 밀(密) (기밀)
보초 초(哨) + 군인 병(兵) (초병)
초소 초(哨) + 곳 소(所) (초소)
독 독(毒) + 해칠 유(毒) + 음식 음(飮) + 음식 식(食) + 물건 물(物) + 공 공(供) + 급할 급(給) (유독음식물공급)
잡힐 포(捕) + 잡힐 로(虜) (포로)
군사 군(軍) + 쓸 용(用) + 물건 물(物) (군용물)
허물 죄(罪) (죄)
법 법(法) + 율 륥(律) (법률)
나라의 중요한 군사 비밀을 빼돌리거나, 보초 서는 군인을 해치거나, 군인들에게 독이 든 음식을 주는 등 군대와 나라의 안전에 직접적이고 아주 심각한 피해를 주는 특정 범죄를 저지른 경우입니다.
🚨
비상계엄이 선포된 경우를 제외하고는
아닐 비(非) + 항상 상(常) + 경계할 계(戒) + 엄할 엄(嚴) (비상계엄)
베풀 선(宣) + 펼 포(布) (선포)
전쟁과 같은 최고 수준의 국가 비상사태가 선포되었을 때입니다.

결론적으로...

헌법 제27조 제2항은, 군인이 아닌 모든 국민은 일반 법원에서 재판을 받을 권리가 있음을 보장하는 조항입니다.

이는 군사재판이 일반 국민에게 확대되는 것을 막아 국민의 인권을 보호하기 위함입니다.

다만, 국가의 안전보장에 직접적으로 관련된 매우 중대한 특정 범죄를 저지르거나, 비상계엄이 선포된 극히 예외적인 상황에서만 군사재판을 받을 수 있도록 하여 국가의 안전과 국민의 인권을 함께 지키고 있습니다.
